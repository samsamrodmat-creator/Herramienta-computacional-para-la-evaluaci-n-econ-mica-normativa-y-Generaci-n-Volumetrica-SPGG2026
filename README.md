Asesorado por: Arq. M.S.c Andrés Obregón López.

Descripción del Proyecto:
Este proyecto calcula la viabilidad de proyectos habitacionales mediante análisis normativo, costos directos/indirectos, usos mixtos, estacionamiento y simulación de rentabilidad.

Uso exclusivamente académico.

Descripción General
Este repositorio contiene las herramientas digitales desarrolladas para la evaluación rápida de proyectos inmobiliarios en etapas tempranas de diseño. El algoritmo principal se ejecuta sobre Rhino/Grasshopper y permite obtener indicadores financieros en tiempo real a partir de volumetrías arquitectónicas.

Contenido del Repositorio
